# Jogo Jó Ken Pô

Exemplo de um jogo Jó Ken Pô desenvolvido em Python. 

O jogador joga cada partida contra a Nicole (computador) e no final é exibido o vencedor.

---

## Configurações

Python 3.8

### Executar o jogo no terminal

```
python3 app.py
```
